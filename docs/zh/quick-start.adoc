只需几分钟，即可启动 WIS2 Downloader 并开始下载数据。

== 前提条件

* https://docs.docker.com/get-docker/[Docker] 20.10+
* https://docs.docker.com/compose/install/[Docker Compose] 2.0+
* 能够访问 WIS2 Global Broker 的网络连接（端口 443）

== 1. 下载并解压

下载 https://github.com/World-Meteorological-Organization/wis2downloader/releases/latest[最新版本] 的压缩包并解压：

[source,bash]
----
tar -xzf wis2downloader-*.tar.gz
cd wis2downloader-*
----

TIP: 若要使用最新开发版本，请克隆代码仓库：`git clone https://github.com/World-Meteorological-Organization/wis2downloader.git`

== 2. 配置

运行安装脚本，生成包含随机密钥的 `.env` 文件：

[source,bash]
----
bash setup.sh
----

该脚本会将 `default.env` 复制为 `.env`，并为 `FLASK_SECRET_KEY` 和 `REDIS_PASSWORD` 生成随机值。请检查 `.env` 文件并根据需要调整配置——默认值适用于单机部署。

IMPORTANT: 请妥善保管 `.env` 文件。该文件包含敏感凭据，不得提交到版本控制系统。

== 3. 安装 Loki Docker 插件

WIS2 Downloader 使用 https://grafana.com/docs/loki/latest/send-data/docker-driver/[Loki Docker 驱动程序] 进行日志传输。请在启动服务前安装该插件：

[source,bash]
----
ARCH=$(uname -m | sed 's/x86_64/amd64/;s/aarch64/arm64/')
docker plugin install grafana/loki-docker-driver:3.6.7-${ARCH} --alias loki --grant-all-permissions
----

验证插件已启用：

[source,bash]
----
docker plugin ls
----

应看到 `loki:latest` 的 `ENABLED` 列显示为 `true`。

NOTE: 如果安装时出现错误，请通过 `uname -m` 确认您的系统架构。最常见的值为 `x86_64`（amd64）和 `aarch64`（arm64）。

== 4. 设置下载目录

容器以非 root 用户身份运行（UID 10001，GID 988）。在 Linux 上，启动前必须配置下载目录权限：

[source,bash]
----
sudo groupadd -g 988 wis2
sudo useradd -u 10001 -g wis2 -M -s /usr/sbin/nologin wis2
mkdir -p downloads
sudo chown wis2:wis2 downloads
----

NOTE: 在 macOS 和 Windows（Docker Desktop）上无需此步骤——Docker Desktop 会自动处理卷权限。

如果无法创建匹配的用户，请参阅管理指南中的 link:admin-guide.html#_volume_permissions[卷权限] 章节，了解基于 ACL 的替代方案。

== 5. 启动服务

[source,bash]
----
docker compose up -d
----

此命令将启动所有服务：订阅管理器、MQTT 订阅者（法国和巴西 Global Broker）、Celery 下载工作进程、Redis、Prometheus、Grafana 和 Loki。

检查所有服务是否正常运行：

[source,bash]
----
docker compose ps
----

等待 `subscription-manager` 健康检查通过（约 30 秒），然后确认其已就绪：

[source,bash]
----
curl http://localhost:5002/health
----

== 6. 创建订阅

=== 通过 Web 界面（推荐）

在浏览器中打开 http://localhost:8080。

使用 *目录视图* 或 *树形视图* 浏览可用的 WIS2 数据集，选择一个主题，指定保存目录，然后点击 *订阅*。

=== 通过 API

[source,bash]
----
curl -X POST http://localhost:5002/subscriptions \
  -H "Content-Type: application/json" \
  -d '{
    "topic": "cache/a/wis2/+/data/core/weather/surface-based-observations/#",
    "target": "surface-obs"
  }'
----

此操作将订阅来自任意数据中心的所有地面观测数据，并将文件保存到 `downloads/surface-obs/`。

== 7. 查看下载内容

文件按日期组织，存放在 `downloads/` 目录下：

[source,bash]
----
ls -la downloads/surface-obs/
----

通过 Grafana 监控下载活动，访问地址：http://localhost:3000（默认登录凭据：`admin` / `admin`）。

IMPORTANT: 首次登录后请立即修改 Grafana 的默认密码。
